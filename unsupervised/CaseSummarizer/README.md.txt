This repository contains an implementation of the paper CaseSummarizer http://www.aclweb.org/anthology/C16-2054

Input : path to a directory containing the documents to be summarized (input_path), path to the directory in which the summaries of the individual files wil be stored (output_path), path to a legal dictionary (dict_path)

Output : Summaries will be stored in the directory specified by output_path

Dependencies : nltk, numpy
